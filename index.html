My first website

TheWorld Wide Web(WWW or simply the web) is an information systemthat enables contentsharing over the Internetthrough user-friendly ways meant to appeal to users beyond IT specialists and hobbyists.It allows documents and other web resourcesto be accessed over the Internet according to specific rules of the Hypertext Transfer Protocol(HTTP).

The original and still very common document type is a web page formatted in Hypertext Markup Language(HTML). This markup language supportsplain text, images,embedded video and audio contents, and scripts (short programs) that implement complex user interaction
